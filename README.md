# 📚 BookSky — Book Management System

A simple and interactive web-based Book Management System where users can add and manage book records dynamically.

---

## 🚀 Features

- Add books using a clean popup form
- Form fields: **Book Title**, **Author**, and **Short Description**
- **Add** button to save and display the book
- **Cancel** button to close the form without saving
- Books displayed dynamically as **cards** using JavaScript DOM manipulation
- Clean and responsive UI built with HTML5 and CSS3

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🖼️ How It Works

1. Click the "Add Book" button to open the form
2. Fill in the "Title", "Author", and "Description"
3. Click "Add" — the book appears as a card on the page
4. Click "Cancel" to close the form without saving
